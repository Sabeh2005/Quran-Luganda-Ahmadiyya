# Quran Luganda Ahmadiyya

The Noble Quran with Luganda and English translations.

This application provides a comprehensive digital Quran experience, featuring:
- **Complete Quran Text**: Arabic text with accurate Luganda and English translations.
- **Multiple Views**: Read by Surah or search for specific verses.
- **Customization**: Dark mode, font size adjustments (Arabic and translation), and theme colors.
- **Navigation**: "Quick Navigation" to jump to any Surah and Verse instantly.
- **Bookmarks**: Save your favorite verses for quick access.

## Project Info

This project is built to provide an accessible and beautiful way to read the Holy Quran in Luganda and English.

## How to run locally

To run this project locally, you need to have Node.js & npm installed.

Follow these steps:

```sh
# Step 1: Clone the repository
git clone https://github.com/Sabeh2005/Quran-Luganda-Ahmadiyya.git

# Step 2: Navigate to the project directory
cd Quran-Luganda-Ahmadiyya

# Step 3: Install the necessary dependencies
npm i

# Step 4: Start the development server
npm run dev
```
